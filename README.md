------------

## Local installation
1- Clone the repository locally
```bash
git clone https://github.com/alexbernardi360/net-rooms.git
```

2- Enter in the main foulder
```bash
cd net-rooms
```

3- Install the dependencies
```bash
npm install
```

4- Run the project
```bash
npm run start
```
